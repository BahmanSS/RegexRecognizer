# RegexRecognizer
## Таблица переходов ДКА:

| Состояние | a | b | * |
|-----------|---|---|---|
| ->q0 | ∅ | ∅ | q1 |
| q1 | ∅ | ∅ | q2 |
| q2 | q3 | q2 | qf |
| q3 | ∅ | q2 | ∅ |
| *qf | ∅ | ∅ | ∅ |

## Диарграмма перехов ДКА:

[![2025-10-14-232416767.png](https://i.postimg.cc/LsDWzkpF/2025-10-14-232416767.png)](https://postimg.cc/TyyQxD1t)
## Регулярное выражение (РВ):

```
** (ab + b)* *
```
## Примеры допустимых строк
```
**ab*
**bb*
**abbb*
**bababbbbab*
**b*
```
