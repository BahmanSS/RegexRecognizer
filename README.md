# RegexRecognizer
## Таблица переходов ДКА:

| Состояние | a | b | * |
|-----------|---|---|---|
| q0 | ∅ | ∅ | q1 |
| q1 | ∅ | ∅ | q2 |
| q2 | q3 | q2 | qf |
| q3 | ∅ | q2 | ∅ |
| qf | ∅ | ∅ | ∅ |

## Регулярное выражение (РВ):

```
** (ab + b)* *
```
## Примеры допустимых строк
```
**ab*
**bb*
**abbb*
**bababbbbab*
**b*
```
